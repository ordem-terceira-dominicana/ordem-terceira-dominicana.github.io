# Pequeno Ofício de Nossa Senhora segundo o Rito Dominicano

Este projeto apresenta uma versão digital, simples e fiel do **Pequeno Ofício de Nossa Senhora**, seguindo a tradição da **Ordem Dominicana**.  
A tradução foi realizada com recurso a IA a partir da versão disponível em:  
https://sacredheartlawrence.org/little-office-of-the-blessed-virgin-mary/  
e complementada com a edição portuguesa do Pequeno Ofício segundo o Rito Dominicano, presente no *Manual da Ordem Terceira de São Domingos* (1949).

O objetivo é oferecer uma experiência limpa, rápida e contemplativa, com as cores dominicanas (branco e preto) e o conteúdo organizado por horas litúrgicas.

---

## ✝️ Objetivo

Criar uma aplicação web minimalista que:

- apresenta automaticamente a Hora litúrgica atual;
- determina o tempo litúrgico (solar e pascal);
- carrega o texto correspondente a partir de ficheiros Markdown;
- expande includes internos;
- aplica a lógica de *aleluia* conforme o tempo;
- mantém o estilo dominicano tradicional.
